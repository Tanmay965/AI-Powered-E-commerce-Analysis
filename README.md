AI Powered E-Commerce Analysis System

## Project Overview

This project analyzes large-scale e-commerce data to extract meaningful business insights using Python, SQL, Machine Learning, and Power BI. It focuses on customer segmentation, churn prediction, sales forecasting, product recommendation, and business intelligence dashboard development.
The system transforms raw transactional data into actionable insights to support data-driven decision making and business optimization.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Scikit-learn, XGBoost)
- SQL (MySQL)
- Machine Learning (K-Means, Random Forest, TF-IDF, Cosine Similarity)
- Power BI
- Excel

## Dataset
- Customers Dataset
- Orders Dataset
- Order Items Dataset
- Payments Dataset
- Products Dataset
- Reviews Dataset
- Sellers Dataset
- Geolocation Dataset

## Workflow
1. Collected and integrated multiple e-commerce datasets
2. Cleaned and preprocessed data using Python (Pandas, NumPy)
3. Performed data transformation and feature engineering
4. Conducted Exploratory Data Analysis (EDA) using Matplotlib
5. Applied SQL queries for business insights extraction
6. Performed RFM-based customer segmentation using K-Means clustering
7. Built machine learning models for:
    - Customer churn prediction
    - Sales forecasting using XGBoost
    - Product recommendation using TF-IDF and cosine similarity
8. Exported processed data to Power BI
9. Developed interactive dashboards for business intelligence

## Key Features
- Customer segmentation using RFM analysis
- Customer churn prediction system
- Sales forecasting for future revenue
- Product recommendation engine
- Revenue and sales trend analysis
- Payment and order behavior analysis
- Interactive Power BI dashboards

## Outcome
Developed a complete AI-powered analytics system that helps businesses understand customer behavior, predict future trends, reduce churn, and improve revenue through data-driven decision making.
